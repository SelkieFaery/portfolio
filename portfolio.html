Web Developing:


1. Registration Form (2018)


This is a simple form I made for my Fundamentals of JavaScript class at UMUC. I wrote the HML, CSS, and JavaScript used in this. It uses JavaScript to validate required fields. This is a simple, functioning form that will email me, if everything is entered correctly. It has several fields, including a radio buttons and checkboxes that must be completed. First name, last name, city, zip code, phone number, and area code must be filled out correctly. State uses a dropdown menu and something must be selected. The email must be a valid email and the confirm email must match. If anything is filled out incorrectly, an alert message states the error and does not allow the form to be submitted.

2. Image Gallery (2018)


This piece is a simple image gallery, also created for my Fundamentals of JavaScript class at UMUC. This was my first project for my class. It is a collaboration between my professor and myself. The HTML and CSS code was provided by my professor and I added the JavaScript that allows the photo gallery to be interactive. Initially, there is no main image, but only a couple of thumbnail images of various flowers on a black background. When moussing over the images, a short, corresponding description of the image displays under the thumbnails. Clicking on a thumbnail will display the image as the main image and turn the header red.

3. Ticket Purchasing (2018)


This is a simple ticket purchasing system, also created for my Fundamentals of JavaScript class. The HTML and CSS was provided by my professor and I added the JavaScript. The system allows a customer to purchase between 1 and 3 tickets for an event. Attempting to purchase more than 3 or less than 1 ticket will return an error. Once the amount is entered, clicking out of the box will update the total cost and display the rest of the form. A name and email address must be entered correctly, or an error will be displayed. A confirmation window displays when purchase is complete. No credit card information is requested, the form does not get submitted anywhere, no tickets are actually purchased.
